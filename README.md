# React Quote Machine

This simple React application generates A Random Quote each time the 'New Quote' Button is clicked.

### Technologies Used

* React
* Jquery
* Webpack

### Getting Started

```
> git clone https://github.com/DKeen0123/ReactQuoteMachine
> cd ReactQuoteMachine
> npm install
> npm start
```
